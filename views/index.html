Welcome to Timestamp Microservice

Please append a human readable date or UNIX Timestamp of the address of this page to use this app.
If the timestamp you entered is valid you'll get a response including both versions of the timestamp you entered.

Example use:

http://fcc-basejump-bararan.c9users.io/22 oct 2012

and

http://fcc-basejump-bararan.c9users.io/1350864000

produce the following output:

{ "unix": 1350864000, "natural": "Mon Jul 25 1983" }
